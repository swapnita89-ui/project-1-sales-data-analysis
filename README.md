# project-1-sales-data-analysis
1. Project Tital - Sales Data Analysis
2. Project Objective - The objective of this project is to analyze sales data to identify key trends, top-performing regions, and business insights that can help                        improve decision-making.
3. Tech Stack - Below is the list of the tools which i used in this project
    * Language: Python (Pandas, NumPy)
    * Visualization: Matplotlib, Seaborn.
    * Database: 
    * Notebook: Kaggle Notebook

4. Dataset Description - The dataset contains 9800 records of sales transactions with features such as:
    * Order ID
    * Order Date
    * Customer Segment
    * Region
    * Product Category
    * Sales
5. Steps Performed - Below is the key points
    * Data loading and inspection
    * Data cleaning (date conversion, duplicates check)
    * Sales trend analysis
    * Visualization using matplotlib
6. Key insights -
    * November has the highest total sales (~350K), indicating strong seasonal demand
    * Certain regions contribute significantly more revenue than others
    * Some transactions show negative profit, indicating potential loss areas
    * Sales distribution shows presence of high-value outliers
7. Visualization -
    * monthly sales trend
    * category-wise sales
    * region-wise sales
8. Conclusion - The analysis highlights seasonal trends and identifies key revenue-driving regions and categories. These insights can help businesses optimize                    inventory, marketing strategies, and pricing decisions.
